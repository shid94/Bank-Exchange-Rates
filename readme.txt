A dynamic bank exchange rates plugin, marquee ticker, flags, sorting, timestamp, and calculator.

== Description ==

Bank Exchange Rates is a lightweight and flexible WordPress plugin designed for banks, forex bureaus, and financial platforms in Uganda and beyond.

It allows administrators to easily manage currency exchange rates and display them on the frontend in a clean, professional forex format (e.g., USD/UGX).

The plugin includes a scrolling marquee ticker, currency flags, automatic sorting, last updated timestamp, and a built-in forex calculator.

== Features ==

* Dynamic currency management from admin panel
* Professional forex format (e.g., USD – Buy/Sell)
* Country flags for each currency
* Smooth scrolling marquee ticker
* Automatic sorting (USD prioritized)
* Last updated timestamp
* Built-in forex calculator
* Responsive and lightweight

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to **Exchange Rates** in the admin menu
4. Add and save your currency rates

== Usage ==

Use the following shortcodes in your pages or posts:

* Display full table:
  [bank_rates]

* Display scrolling ticker:
  [bank_rates_marquee]

== Frequently Asked Questions ==

= Can I change the base currency? =
Currently, the plugin is optimized for UGX as the base currency, but it can be modified in code.

= Does it support live exchange rates? =
No, rates are manually entered. API integration can be added if needed.

= Can I customize the design? =
Yes, you can modify the CSS file located in `assets/css/style.css`.
